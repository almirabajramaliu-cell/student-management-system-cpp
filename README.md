# Sistemi i Menaxhimit të Studentëve (C++)

## Përshkrimi
Ky është një aplikacion konsolë për menaxhimin e studentëve i zhvilluar në C++.
Projekti demonstron punën në grup duke përdorur GitHub, përfshirë
degëzimin (branches), commit-et dhe pull request-et.

## Funksionalitetet
- Shtimi i studentëve
- Shfaqja e listës së studentëve
- Kërkimi i studentëve
- Ruajtja dhe ngarkimi i të dhënave nga skedari

## Teknologjitë
- C++
- Visual Studio
- GitHub

## Bashkëpunimi në ekip
Secili anëtar i ekipit ka punuar në një funksionalitet të veçantë duke përdorur
degë (feature branches). Të gjitha ndryshimet janë rishikuar dhe bashkuar
në degën kryesore (main) përmes pull request-eve.
